# История изменений

Все заметные изменения в Filmber документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/).

## [2.9.0] - 2026-01-16 — Уведомления о фильмах и сериалах

### Добавлено
- Система уведомлений о новинках кино через Telegram-бот
- Уведомления о премьерах в кинотеатрах (по регионам US/RU)
- Уведомления о цифровых релизах фильмов
- Уведомления о новых сезонах сериалов из списка «Смотрел»
- Уведомления о новых сериях с задержкой 7 дней на озвучку
- Автоматическое перемещение сериалов в «В планах» при обнаружении нового сезона
- Админ-панель для управления уведомлениями и просмотра статистики
- Настройки уведомлений в профиле пользователя

### Изменено
- Расширена схема базы данных для отслеживания сериалов
- Добавлены cron-задачи для синхронизации и отправки уведомлений
- Добавлены rate limits и silent hours (23:00-08:00 MSK)

## [2.8.0] - 2026-01-15 — Полноэкранные детали фильма

### Добавлено
- Полноэкранный режим просмотра деталей фильма с параллакс-эффектом
- Закрытие свайпом вниз в модальном окне фильма
- Интеграция с полноэкранным режимом Telegram
- Модальное окно трейлера с плавными анимациями
- Аккордеон сезонов и эпизодов для сериалов

### Изменено
- Переработан дизайн карточки фильма на полноэкранный режим
- Оптимизировано кэширование фильмов (один запрос на локаль)
- Улучшена дедупликация запросов к API

### Исправлено
- Исправлена ошибка рендеринга жанров (React error #31)
- Исправлена гидратация сторов

## [2.7.0] - 2026-01-15 — Расширенные фильтры поиска

### Добавлено
- Быстрые фильтры: тип контента, язык оригинала, жанры, рейтинг
- Фильтр по продолжительности фильма/сериала
- Поддержка фильтрации старых данных по названию жанра

### Изменено
- Переработано меню фильтров с раскрывающимися секциями
- Улучшена совместимость фильтров со старыми данными в кэше

### Исправлено
- Исправлена фильтрация по жанрам для фильмов из разных источников
- Исправлена фильтрация по языку оригинала
- Исправлена ошибка гидратации во вложенных кнопках

## [2.6.0] - 2026-01-13 — Плавные анимации свайпов

### Изменено
- Плавная анимация ухода карточки при свайпе с вращением
- Эффект глубины стека: карточки позади меньше и с мягкими тенями
- Плавное появление новых карточек снизу

### Исправлено
- Исправлено «метание» карточки при быстрых кликах на кнопки

## [2.5.0] - 2026-01-13 — Страница «Что нового»

### Добавлено
- Страница «Что нового» с историей обновлений
- Названия релизов для удобной навигации

### Изменено
- Переработан формат списка изменений для пользователей

## [2.4.0] - 2026-01-13 — Сериалы и оптимизация

### Добавлено
- Поддержка сериалов с информацией о сезонах и эпизодах
- Улучшена загрузка изображений для регионов с ограничениями

### Изменено
- Обновлённый дизайн карточек фильмов с новыми иконками
- Улучшена читаемость текста и единообразие интерфейса
- Оптимизирована работа с базой данных

### Исправлено
- Улучшена производительность загрузки списков
- Исправлены мелкие ошибки интерфейса

## [2.3.0] - 2026-01-12 — Похожие фильмы и постеры

### Добавлено
- Раздел «Похожие фильмы» на странице фильма
- Локальное кэширование постеров для быстрой загрузки
- Команда /bug в боте для отправки сообщений об ошибках

### Изменено
- Улучшена логика напоминаний о просмотре

## [2.2.0] - 2026-01-12 — Альтернативные базы данных

### Добавлено
- Рейтинги Кинопоиска и IMDB на карточках фильмов
- Ссылки на фильмы в Кинопоиске и IMDB

### Исправлено
- Корректное отображение мгновенных совпадений
- Исправлена работа кнопки очистки в поиске

## [2.1.0] - 2026-01-12 — Улучшение поиска и рекомендаций

### Добавлено
- Переключатель языка интерфейса
- Кнопка поддержки проекта
- Хронологическая сортировка фильмов одной франшизы

### Изменено
- Улучшен алгоритм поиска фильмов
- Поиск теперь находит фильмы на обоих языках

### Исправлено
- Корректная сортировка эпизодов франшизы
- Правильная обработка буквы «ё» в поиске

## [2.0.0] - 2026-01-11 — Telegram-бот и новый интерфейс

### Добавлено
- Telegram-бот для управления списками и получения уведомлений
- Система рекомендаций на основе очереди
- Настройки колоды для персонализации подбора

### Изменено

- Новые анимированные кнопки и компоненты

## [1.1.0] - 2026-01-10 — Режим соло и аналитика

### Добавлено
- Режим соло для индивидуального просмотра фильмов
- Раскрывающиеся описания на карточках фильмов
- Аналитика использования приложения

### Исправлено
- Улучшена стабильность работы приложения

## [1.0.0] - 2026-01-08 — Первый релиз

### Добавлено
- Парный подбор фильмов с синхронизацией в реальном времени
- Авторизация через Telegram
- Списки фильмов с оценками и статусом просмотра
- Система комнат с QR-кодами и PIN-кодами
- Поддержка русского и английского языков
- Реферальная система с отслеживанием приглашений
